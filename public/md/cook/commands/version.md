# Cook version

Show Cook CLI version information and banner.

## Usage

```bash
cook version
```

## Description

Displays the Cook CLI version information along with the ASCII art banner. This command helps you verify which version of Cook CLI you're running.

## Example

```bash
$ cook version

  _________  ____  __ __
 / ___/ __ \/ __ \/ //_/
/ /__/ /_/ / /_/ / ,<
\___/\____/\____/_/|_|


COOK_VERSION 1.0.0 (alpha)
```

## Options

This command has no additional options.

## Notes

- The version command is useful for troubleshooting and ensuring you have the correct version installed
- The ASCII banner provides visual confirmation that Cook CLI is working properly
